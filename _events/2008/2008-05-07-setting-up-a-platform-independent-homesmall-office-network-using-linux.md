This was intended to provide attendees with an idea of which software products would best help them run a multi-os home/small office network.
- Samba
- Fileshaing
- Printsharing
- IMAP
- Email synchronization
- Gnome-RDP
- RDP client (rdesktop)
- VNC client (tightvnc)
- BackupPC
- With special guest lecturer
